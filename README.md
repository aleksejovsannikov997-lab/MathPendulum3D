# 3D Математический маятник в Unity

## Описание
Симуляция математического маятника в Unity 6 (3D, URP).  
Движение рассчитывается по формуле `θ'' = -(g/L) * sin(θ)` методом Эйлера.

## Как запустить
1. Скачать репозиторий (Download ZIP или `git clone`)
2. Открыть проект в Unity 6 LTS
3. Открыть сцену `Assets/Scenes/PendulumScene.unity`
4. Нажать Play

## Параметры
Длина, начальный угол и гравитация настраиваются в инспекторе объекта `Pivot`.
